# GitHub Community Master Prompts

## Repo genesis prompt

```text
Create a Sonic-Forage repo from this idea. Include README, docs, prompt pack, payload examples, tag taxonomy, `.env.example`, verifier, Pages deck, issue templates, and closed-gate approval ledger. Make it community forkable and source-backed where factual.
```

## Issue-to-payload prompt

```text
Convert this GitHub issue into a JSON payload with: lane, requested_output, safety_gates, assets_needed, endpoint_contracts, verification_commands, tags, and next_safe_increment. Do not include secrets or real endpoint URLs.
```

## PR review prompt

```text
Review this PR for: community usefulness, verifier coverage, tag consistency, no secrets, no private media, no real endpoint URLs, no overclaims, and whether all external actions remain closed_until_human_yes.
```

## Arcade downtime prompt

```text
Services are down. Generate a fun status note that tells users to play Strawberry AGI Snake while agents recover. Include no false uptime claims. Link to the live arcade and show current known status if provided.
```
