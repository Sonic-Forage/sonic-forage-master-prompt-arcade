## 📖 DJ Terminology Quick Reference

### Essential Terms
| Term | Meaning | DJ Context |
|------|---------|------------|
| **BPM** | Beats Per Minute | Track speed — house ~128, techno ~130, DnB ~174 |
| **Drop** | Bass kicks back in after breakdown | The moment the crowd explodes |
| **Build-up** | Rising tension before drop | Snare rolls, filter sweeps, risers |
| **Breakdown** | Reduced energy section | Stripped-back, atmospheric |
| **Phrasing** | Musical sentence structure | 8/16/32 bar blocks |
| **Cue** | Starting point marker | Where the track begins playing |
| **Trim/Gain** | Volume matching | Keep levels consistent across tracks |
| **EQ** | Equalization | Low/Mid/High frequency control |
| **Filter** | Frequency sweep | Low-pass = muffled, high-pass = tinny |
| **Reverb** | Spatial echo effect | Adds depth and space |
| **Delay** | Timed echo | Creates rhythmic patterns |
| **Loop** | Repeating section | Keeps a groove going |
| **Acapella** | Vocals only | For mashups and overlays |
| **ID** | Unreleased/unknown track | "What's the ID?" = what song is that? |
| **Rinse** | Play extensively | "Rinse this tune" = play it a lot |
| **Wheel/Rewind** | Rewind and replay | Crowd reaction so good you play it again |
| **Mashup** | Two tracks combined | Vocals from A + beats from B |
| **Edit** | Modified version | Custom arrangement or bootleg |

| Genre | BPM Range | Vibe | Signature Sound |
|-------|-----------|------|-----------------|
| **House** | 120-130 | Groovy, soulful | 4/4 kick, piano stabs |
