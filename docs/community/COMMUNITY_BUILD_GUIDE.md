# Community Build Guide

## How to contribute

1. Fork the repo.
2. Pick a small lane: prompt, payload, tag, workflow, UI card, arcade polish.
3. Keep external actions closed.
4. Run `python3 scripts/verify.py`.
5. Open a PR with screenshots for UI/game changes.

## Good first forage ideas

- Add a new voice profile card.
- Add a new workflow payload.
- Add a new Unicode status card.
- Add a new arcade enemy type that does not require external services.
- Add a new GitHub issue prompt.
